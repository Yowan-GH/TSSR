# Le routage

## Definition

**Le routage permet la communication entre plusieurs réseaux logiques**.  
Il peut être statique ou dynamique.  

Une route est constituée de :  
- Adresse réseau de destination
- Masque de sous réseau
- Adresse de passerelle
